JavaScript Closures: A Deep Dive

Intermediate ⏱ 15 min read 3/5 sections
What are Closures?

A closure is a feature in JavaScript where an inner function has access to the outer (enclosing) function's variables. This includes:

  • Variables from the outer function's scope
  • Global variables
  • The outer function's parameters
Interactive Example:
function outerFunction(x) {
// This is the outer function's scope
function innerFunction(y) {
// This inner function has access to 'x'
return x + y;
}
return innerFunction;
}
const addFive = outerFunction(5);
console.log(addFive(3)); // Output: 8
💡
AI Insight

Think of closures like a backpack that a function carries. Even when the function travels to different parts of your code, it still has access to everything in its backpack (the outer scope variables).
